Para guardar un archivo en Visual Studio Code, puedes hacerlo de varias formas:

✅ Opción 1: Atajo de teclado

  • Windows/Linux: Ctrl + S

  • Mac: Cmd + S

✅ Opción 2: Desde el menú

  1. Haz clic en File (Archivo) en la barra superior.

  2. Selecciona Save (Guardar) o Save As... (Guardar como...).

✅ Opción 3: Clic derecho

  • Haz clic derecho dentro del archivo abierto y selecciona Save.

💡 Si es un archivo nuevo (sin nombre aún), te pedirá:

  • Nombre del archivo (ej: mi_programa.py, index.html, etc.)

  • Carpeta o ubicación donde quieres guardarlo.

¿Quieres que te muestre cómo guardar un tipo de archivo específico, como HTML, Python, LaTeX, etc.?

Perfecto, si estás trabajando con varios archivos abiertos en Visual Studio Code y solo quieres guardar uno, aquí te dejo cómo hacerlo:

🔹 OPCIÓN 1: Atajo solo para el archivo activo

Presiona:

  • Ctrl + S (Windows/Linux)

  • Cmd + S (Mac)

Esto solo guarda el archivo que estás viendo actualmente en el editor, no los demás.

🔹 OPCIÓN 2: Guardar desde el menú

  1. Haz clic en la pestaña del archivo que quieres guardar (asegúrate de que esté activo).

  2. Luego ve a:

    • Archivo → Guardar

    • O clic derecho sobre el archivo abierto → Guardar

🔹 ¿Cómo saber si un archivo no está guardado?

  • Si en la pestaña del archivo aparece un ● o un nombre en cursiva, significa que tiene cambios sin guardar.
